AI Chat Widget Test - Feature #88

Feature ID: #88 - Add AI chat widget to page

Verify users can add AI chat widget for AI conversations

Test Steps:

  1. Open browser DevTools (F12) and check for console errors
  2. Click the "Add Widget" button on the page
  3. Select "AI Chat Widget" from the widget type selector
  4. Verify the AI chat widget is created with a default title "AI Chat"
  5. Verify the widget shows a chat interface with input field and send button
  6. Verify the widget prompts for provider setup (OpenAI/Straico)
  7. Verify the widget configuration modal allows setting API keys and models
