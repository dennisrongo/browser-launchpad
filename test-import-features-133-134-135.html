Feature #134: Validate Imported JSON Structure

Status: Not started

Test Invalid JSON Syntax:

Create a file with invalid JSON (e.g., missing closing brace)
Attempt to import it
Verify validation error appears: "Invalid JSON syntax"

Test Missing Version:

Create JSON without version field
Attempt to import it
Verify error: "Invalid file: missing version information"

Test Incompatible Version:

Create JSON with version "2.0.0"
Attempt to import it
Verify error: "Incompatible version: 2.0.0"

Test Missing Data Section:

Create JSON with version but no data field
Attempt to import it
Verify error: "Invalid file: missing or invalid data section"

Test Valid JSON:

Use properly formatted export file
Verify import succeeds

Status: Not started

Test Steps:
