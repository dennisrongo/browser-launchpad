Feature #106: Model Switching Clears Chat (with Confirmation)

Test Overview

This test verifies that switching AI providers or models shows a confirmation dialog and clears chat history when confirmed, but preserves chat history when canceled.

1. Provider Switch Confirmation

  • Send messages in AI chat (OpenAI provider)
  • Click edit on widget
  • Change provider to Straico
  • Verify confirmation dialog appears
  • Confirm the change
  • Verify chat history is cleared
  • Verify new provider is active

2. Model Switch Confirmation (OpenAI)

  • Send messages in AI chat with GPT-3.5
  • Click edit on widget
  • Change model to GPT-4
  • Verify confirmation dialog appears
  • Confirm the change
  • Verify chat history is cleared
  • Verify new model is active

3. Cancel Preserves Chat

  • Send messages in AI chat
  • Click edit on widget
  • Change to different model
  • Verify confirmation dialog appears
  • Cancel the change
  • Verify chat is preserved
  • Verify original model is still selected

4. Warning Text Display

  • Verify orange warning appears when messages exist
  • Warning text: "⚠️ Changing provider will clear chat history"
  • Warning text: "⚠️ Changing model will clear chat history"

5. Empty Chat - No Confirmation

  • Open AI chat widget with no messages
  • Click edit on widget
  • Change provider or model
  • Verify NO confirmation dialog appears
  • Change happens immediately

Manual Testing Instructions

  1. Run npm run build
  2. Load extension in Chrome
  3. Create an AI Chat widget
  4. Configure with OpenAI API key and select a model
  5. Send a few messages
  6. Click Edit on the widget
  7. Change the provider dropdown (OpenAI → Straico or vice versa)
  8. CONFIRM: Confirmation dialog appears with "Clear Chat History?" title
  9. Click "Clear & Switch"
  10. CONFIRM: Widget config is saved with empty messages array
  11. CONFIRM: Chat history in widget shows empty state
  12. Repeat steps 4-10 but click "Cancel" instead
  13. CONFIRM: Chat history is preserved, dropdown selection reverts
