Feature #149: User-Friendly Error Messages

Testing that all error messages are clear, actionable, and helpful to users.

Criteria Checklist

⬜
Clear Messages: All error messages are in plain language, not technical jargon.
⬜
Actionable: Each error suggests what the user should do next.
⬜
Contextual: Errors explain what went wrong and where.
⬜
User-Friendly: No stack traces or technical codes visible to users.

1. API Key Errors

1.1 Missing OpenAI API Key Pending

Should show: "Configure your API key to start chatting" and provide clear instructions.

1.2 Invalid API Key Format Pending

Should show: "OpenAI API key must start with 'sk-'" with actionable guidance.

1.3 Invalid API Key (401) Pending

Should show: "OpenAI: Invalid API key. Please check your API key in widget settings."

1.4 Insufficient Quota Pending

Should show: "OpenAI: Insufficient quota. Please check your billing details."

1.5 Model Not Found Pending

Should show: "OpenAI: Model not found. Please select a valid model in settings."

2. Network Errors

2.1 Network Failure Pending

Should show: "OpenAI: Network error. Please check your internet connection and try again."

2.2 Rate Limit (429) Pending

Should show: "Rate limit exceeded. Please wait X seconds before trying again." with retry information.

2.3 Service Error (500) Pending

Should show: "OpenAI: Service error. Please try again later."

3. Weather Widget Errors

3.1 Missing API Key Pending

Should show: "Please add your OpenWeatherMap API key in widget settings"

3.2 Invalid API Key Pending

Should show: "Invalid API key. Please check your OpenWeatherMap API key"

3.3 City Not Found Pending

Should show: 'City "NotRealCity123" not found. Please check the city name'

4. Settings and Validation Errors

4.1 Import Invalid File Pending

Should show: "Invalid file: not a valid JSON object" with helpful details.

4.2 Import Wrong Version Pending

Should show: "Incompatible version: X.X.X. This extension supports version 1.0.0"

4.3 Import File Too Large Pending

Should show: "File is too large (X.XXMB). Maximum size is 10MB."

4.4 Storage Quota Exceeded Pending

Should show: "Storage quota exceeded. The data is too large to import. Try exporting some data first or use merge mode."

4.5 Invalid JSON Syntax Pending

Should show: "Invalid JSON syntax. Please check the file format."

5. Straico Provider Errors

5.1 Straico Invalid Key Pending

Should show: "Straico: Invalid API key. Please check your API key in widget settings."

5.2 Straico Access Denied Pending

Should show: "Straico: Access denied. Please verify your API key permissions."

5.3 Straico Insufficient Credits Pending

Should show: "Straico: Insufficient credits. Please top up your account."
